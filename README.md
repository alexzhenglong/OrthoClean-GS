# OrthoClean-GS

[![Project Page](https://img.shields.io/badge/Project-Page-blue?style=for-the-badge&logo=github)](https://alexzhenglong.github.io/OrthoClean-GS/)

Dynamic-aware 3D Gaussian Splatting for artifact-free UAV orthophoto (DOM) generation. Removes moving vehicles via 2D semantic priors and 3D-2D consistency, producing clean, ghost-free orthophotos in dynamic urban scenes.

> **📢 Update:** The code and datasets will be officially open-sourced soon. Currently, you can visit our [Project Page](https://alexzhenglong.github.io/OrthoClean-GS/) to explore the interactive visual results of our vehicle removal pipeline.
